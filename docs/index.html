Before you start

  1. Use Chrome on a desktop computer or laptop.
  2. Connect the LILYGO T-Encoder Pro board to your compter with a USB cable.
  3. If the port does not appear, hold BOOT button on the back of the board (not the knob), tap the reset button, then connect.
  4. Do not hold the knob while plugging in the USB cable - GPIO 0 enters download mode.

Install flashes the latest FlightScnr-tencoder-pro-merged.bin from GitHub Releases at offset 0x0.
