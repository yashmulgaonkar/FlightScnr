Erase entire flash?

This will permanently erase all 16 MB of flash memory on the ESP32. Firmware, Wi‑Fi credentials, saved settings, and route cache will be deleted.

The board will not run FlightScnr until you flash firmware again.

Before you start

  1. Use Chrome on a desktop computer or laptop.
  2. Connect the LILYGO T-Encoder Pro board to your compter with a USB cable.
  3. If the port does not appear, hold BOOT button on the back of the board (not the knob), tap the reset button, then connect.
  4. Do not hold the knob while plugging in the USB cable - GPIO 0 enters download mode.

Install updates the app at offset 0x10000 and keeps Wi‑Fi and saved settings. Upload a merged .bin only for a full factory flash at 0x0. Erase flash wipes the entire chip (same as a blank factory part) — use only when you want to clear everything before reflashing.
